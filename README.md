# ⚡ Quick Copy-as-Link-Address - Browser Extension

> **Hover + `Ctrl + Q` = Link Copied.** That's it.

### No more right-click → copy as link address nonsense

**Instant link copying for impatient people.**

- ✅ Hover over any link
- ✅ Press `Ctrl + Q`
- ✅ Done. Your link is copied.

Works everywhere. Saves time. Zero complications.

**For when speed matters.** ⚡

## 📦 Installation (Any Chromium Browser)

This extension works on all Chromium-based browsers like Chrome, Brave, and Microsoft Edge.

### 🔧 Steps to Install

1. Download or clone this repository:

   ```bash
   https://github.com/yuva-raj444/quick_copy-as-link-address.git
   ```

2. Open your browser and go to the extensions page:

   * Chrome → `chrome://extensions/`
   * Brave → `brave://extensions/`
   * Edge → `edge://extensions/`

3. Enable **Developer Mode** (toggle at top right)

4. Click **Load unpacked**

5. Select the project folder (`quick_copy-as-link-address`)

---

### ✅ Usage

* Hover your cursor over any link
* Press **Ctrl + Q**
* The link will be copied instantly to your clipboard

---

### ⚠️ Notes

* Make sure clipboard permissions are allowed
* If using Brave, disable Shields on sites if needed
* Shortcut keys may vary depending on browser conflicts

---

### 🔄 Updating the Extension

After making changes to the code:

1. Go back to the extensions page
2. Click **Reload** on your extension
